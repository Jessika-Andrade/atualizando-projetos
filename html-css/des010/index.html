Tipos de Mídia

Tag <img>

Foi utilizado a tag e o caminho src para mostrar o caminho, alt para descrever o que há na imagem e width para ajustar a largura.

Tag <audio>

Foi utilizado a tag src para mostrar o caminho e os controles:
autoplay = toca sozinho.
muted = para vir silencioso.
loop = para recomeçar quando chegar ao fim.
E também há uma observação caso o áudio não funcione no navegador, o usuário será informado.

Tag <video>

Foi utilizado a tag src para mostrar o caminho. E usado width para diminuir o tamanho do arquivo.

Tag <iframe>

Foi copiado o iframe diretamente do site. Nesse caso, o maps. Também foi alterado a largura e a altura.
